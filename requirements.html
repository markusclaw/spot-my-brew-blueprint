MVP scope, feature list, and phasing

MVP Scope

Phase 0 (Seed) Blueprint — All requirements marked DRAFT pending team refinement

The MVP focuses on core beer discovery and community ratings. The following features are in-scope for v1.0:

Core Features (V1.0)

Beer Catalog & Search

Searchable, filterable database of craft beers available in Mexico. Filter by style, ABV, brewery, region.

Community Ratings

Users can rate and review beers (1-5 stars + text review). Aggregate ratings displayed on beer detail pages.

Where to Buy/Find

Store locations and availability data. Which bars, shops, or breweries carry each beer. User-contributed updates.

Brewery Profiles

Detailed brewery information: origin story, location, beer lineup, social links. Community-editable profiles.

User Profiles

Simple user profiles: username, bio, favorite beers, reviews, follow other enthusiasts.

Basic Authentication

Email or GitHub login. No complex OAuth setup required.

Deferred (V2+)

These are intentionally scoped out of the MVP. They may be revisited post-launch:

  • •Social features (follow, recommendations, feed)
  • •User badges/achievements
  • •Brewery owner portal & analytics
  • •Advanced maps integration (heat maps, routes to stores)
  • •Mobile native app (web MVP first)
  • •Monetization (ads, sponsored listings, premium tiers)

Definition of Done (V1.0)

MVP is ready to launch when:

  • ✓Beer catalog with ≥500 beers (manual + API seeding)
  • ✓Users can create accounts, search, rate, and review
  • ✓Where-to-buy data for top 100 beers (manual curation initially)
  • ✓Brewery profiles for all major Mexican craft breweries
  • ✓Mobile-responsive design (mobile-first)
  • ✓Search and filtering working
  • ✓Basic analytics and error monitoring
